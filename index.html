Seletores CSS

Formas de selecionar elementos para estilização.

  • Seletores de tipo (element)
  • Seletores de classe (.class)
  • Seletores de ID (#id)
  • Seletores de atributo ([attr])
  • Pseudo-classes (:hover)

Propriedades de Layout

Controle a disposição dos elementos na página.

  • display (block, flex, grid)
  • position (relative, absolute)
  • flexbox (flex-direction, justify-content)
  • grid (grid-template, gap)
  • margin e padding

Cores & Tipografia

Estilização de texto e cores.

  • color e background-color
  • font-family e font-size
  • line-height e letter-spacing
  • text-align e text-decoration
  • gradients e rgba/hsla

Efeitos & Animações

Propriedades para criar interações e movimentos.

  • transition (property, duration)
  • animation e @keyframes
  • transform (rotate, scale)
  • box-shadow e text-shadow
  • filter (blur, contrast)

Design Responsivo

Técnicas para layouts adaptáveis.

  • @media queries
  • viewport meta tag
  • unidades relativas (rem, vw)
  • flexbox e grid responsivos
  • mobile-first design

Sobre CSS

  • CSS significa Cass
  • Composta por regras: tem pelo menos um seletor; possuem propriedade/valores
  • Técnicas modernas de estilização
  • Boas práticas e performance
  • Organização por categorias funcionais
