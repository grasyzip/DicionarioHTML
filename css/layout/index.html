Propriedades de Layout

Controle a disposição dos elementos na página com estas propriedades.

Nenhuma propriedade encontrada com ""

Tente outro termo ou verifique a ortografia.

display: block;
O elemento ocupa a largura total disponível e inicia em uma nova linha.
div { display: block; }
display: inline;
O elemento ocupa apenas a largura necessária e não quebra a linha.
span { display: inline; }
justify-content
Alinha os itens flex no eixo principal (horizontal por padrão).
justify-content: center;
align-items
Alinha os itens flex no eixo cruzado (vertical por padrão).
align-items: stretch;
flex-direction
Define a direção do eixo principal (linha ou coluna).
flex-direction: column;
grid-template-columns
Define o número e tamanho das colunas no layout de grid.
grid-template-columns: 1fr 1fr;
grid-gap
Cria espaçamento entre as linhas e colunas do grid.
grid-gap: 20px;
grid-area
Atribui um nome a uma área do grid para fácil posicionamento.
grid-area: header;
display: flex;
Habilita o contexto flexbox para o container, alinhando itens em uma direção.
.container { display: flex; }

display: grid;
Habilita o contexto grid para o container, para criar layouts complexos em duas dimensões.
.container { display: grid; }
