Design Responsivo

Técnicas para criar layouts que se adaptam a diferentes tamanhos de tela.

Nenhum seletor encontrado com ""

Tente outro termo ou verifique a ortografia.

Meta Viewport
A tag `<meta name="viewport">` é essencial para que a página se adapte ao tamanho da tela em dispositivos móveis.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Media Queries
Permitem aplicar estilos CSS apenas quando certas condições são verdadeiras, como a largura da tela.
@media (max-width: 768px) {
  body { font-size: 1.2rem; }
}

Unidades Relativas
Unidades como `%`, `em`, `rem`, `vw` e `vh` se ajustam ao tamanho do elemento pai ou da viewport.
width: 50%;
font-size: 1.2rem;
height: 100vh;

Demonstração de Unidades Relativas

O texto dentro da caixa tem um tamanho de fonte de `2vw` e o quadrado tem largura de `50%` e altura de `10vh`.

Este texto muda de tamanho com a janela.
Imagens Flexíveis
Garantir que as imagens se ajustem à largura do contêiner.
img {
  max-width: 100%;
  height: auto;
}
