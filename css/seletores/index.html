Seletores CSS

Seletores são padrões que permitem selecionar elementos HTML para aplicar estilos.

Nenhum seletor encontrado com ""

Tente outro termo ou verifique a ortografia.

Seletor de Tipo (p)
Seleciona todos os elementos de um tipo específico.
p { color: blue; }

Demonstração de Seletor de Tipo

Este parágrafo foi estilizado com o seletor de tipo `p`.

Este texto está dentro de uma `div` e não foi alterado.

Este é outro parágrafo que também foi afetado.

Seletor de Classe (.classe)
Seleciona elementos com um atributo `class` específico.
.destaque { background: yellow; }

Demonstração de Seletor de Classe

Este parágrafo não tem a classe.

Este parágrafo foi destacado com a classe `.destaque`.

Este é um span com a mesma classe.
Seletor de ID (#id)
Seleciona um único elemento com um atributo `id` específico.
#header { height: 100px; }

Demonstração de Seletor de ID

Este é o elemento com ID `header-exemplo`.
Seletor de Atributo ([atributo="valor"])
Seleciona elementos com um atributo e valor correspondentes.
[type="text"] { border: 1px solid #ccc; }

Demonstração de Seletor de Atributo

O campo de texto abaixo tem borda, mas o de e-mail não.

Atributo com valor ([target="_blank"])
Seleciona elementos com o atributo e valor especificado, usado para abrir links em nova aba.
a[target="_blank"]::after { content: " ↗"; }

Demonstração de Seletor de Atributo com Valor

O link abaixo com o símbolo ↗ abrirá em uma nova aba.

Pseudo-classe (:hover)
Seleciona um elemento quando o mouse passa sobre ele.
a:hover { text-decoration: underline; }

Demonstração de Pseudo-classe :hover

Passe o mouse sobre o botão para ver a cor mudar.

Pseudo-classe (:nth-child())
Seleciona elementos com base em sua posição dentro de um grupo.
li:nth-child(odd) { background: #f5f5f5; }
