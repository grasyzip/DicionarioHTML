Tags para Listas e Tabelas

Estas tags são usadas para criar listas ordenadas, não ordenadas e tabelas em páginas HTML.

Nenhuma tag encontrada com ""

Tente outro termo ou verifique a ortografia.

<ul>
Define uma lista não ordenada (com marcadores).
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
Define uma lista ordenada (com números ou letras).
<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>
<li>
Define um item de lista (usado dentro de <ul> ou <ol>).
<li>Item da lista</li>
<dl>, <dt>, <dd>
Define uma lista de descrição (dl), com termos (dt) e descrições (dd).
<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação para páginas web</dd>
  <dt>CSS</dt>
  <dd>Linguagem para estilização de páginas web</dd>
</dl>
<table>
Define uma tabela.
<table>
  <tr><th>Cabeçalho</th></tr>
  <tr><td>Dado</td></tr>
</table>
<tr>
Define uma linha de tabela.
<tr><td>Dado</td></tr>
<td>
Define uma célula de dados de tabela.
<td>Conteúdo da célula</td>
<th>
Define uma célula de cabeçalho de tabela.
<th>Título da coluna</th>
