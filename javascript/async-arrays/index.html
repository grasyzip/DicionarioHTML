    Métodos de Arrays

    push()
    Adiciona um ou mais elementos ao final de um array e retorna o novo comprimento.
    array.push(elemento1, elemento2, ...);

    Demonstração do método push()

    // Array inicial
let frutas = ['maçã', 'banana'];
frutas.push('laranja');
console.log(frutas);
    pop()
    Remove o último elemento de um array e retorna esse elemento.
    let ultimoElemento = array.pop();

    Demonstração do método pop()

    // Array inicial
let numeros = [10, 20, 30, 40, 50];
let removido = numeros.pop();
console.log(removido);
    unshift()
    Adiciona elementos no início do array.
    array.unshift(elemento)
    let cores = ['verde','azul'];
cores.unshift('vermelho');
console.log(cores);
    splice()
    Adiciona, remove ou substitui elementos em qualquer posição.
    array.splice(início, quantidade, ...itens)
    let letras = ['a','b','c','d'];
letras.splice(1,2);
console.log(letras);
    Outros Métodos
    map, filter, reduce, concat, slice, includes, indexOf, join.
    array.map(fn); array.filter(fn); array.reduce(fn, init);
    let arr1 = [1,2];
let arr2 = [3,4];
console.log(arr1.concat(arr2));
console.log([1,2,3].map(n => n*2));
