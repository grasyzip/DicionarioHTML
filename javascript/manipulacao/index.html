Exemplos de Manipulação

// Selecionar elementos
        const div = document.getElementById('conteudo');
        const paragrafo = document.querySelector('#conteudo p');

        // Alterar conteúdo
        div.innerHTML = '<p>Novo conteúdo</p>';

        // Adicionar elemento

        novoElemento.textContent = 'Texto do span';
        div.appendChild(novoElemento);

        // Alterar estilo

Promises e Async/Await

Trabalhando com operações assíncronas.

// Criando uma Promise
const fetchData = () => {
return new Promise((resolve, reject) => {
setTimeout(() => {
resolve("Dados recebidos!");
}, 2000);
});
};

// Usando Async/Await
async function getData() {
try {
const result = await fetchData();
console.log(result);
} catch (error) {
console.error(error);
}
}

Selecionando e modificando elementos do DOM.

elemento.innerHTML = "<strong>Texto em negrito</strong>";

});
Recursos do ES6+

Novos recursos introduzidos no ECMAScript 2015 e além.

// Template literals
const nome = "Carlos";
const mensagem = `Olá, ${nome}!`;

// Destructuring assignment
const [a, b] = [1, 2];
const {x, y} = {x: 10, y: 20};

// Spread operator
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

// Parâmetros padrão
function saudar(nome = "Visitante") {
return `Olá, ${nome}!`;
