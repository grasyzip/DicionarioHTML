Formulários

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online.

Nenhuma tag encontrada com ""

Tente outro termo ou verifique a ortografia.

<!DOCTYPE>
Define o tipo de documento e versão do HTML.
<!DOCTYPE html>
<html>
Elemento raiz que contém todo o documento HTML.
<html lang="pt-br">...</html>
<head>
Contém metadados e informações sobre o documento.
<head><title>Título</title></head>
<body>
Contém todo o conteúdo visível da página.
<body>Conteúdo aqui...</body>
<header>
Define o cabeçalho de uma seção ou página.
<header><h1>Título</h1></header>
<main>
Define o conteúdo principal único do documento.
<main>Conteúdo principal...</main>
<footer>
Define o rodapé de uma seção ou página.
<footer>Informações de rodapé...</footer>
