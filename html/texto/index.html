Formatação de Texto

Estas tags são usadas para estruturar e formatar conteúdo textual em páginas HTML.

Nenhuma tag encontrada com ""

Tente outro termo ou verifique a ortografia.

<h1> a <h6>
Cabeçalhos/títulos, onde h1 é o mais importante (maior) e h6 o menos importante (menor).
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção</h3>

Título Principal

Subtítulo

Seção

<p>
Define um parágrafo de texto. Os navegadores adicionam automaticamente uma margem antes e depois.
<p>Este é um parágrafo de texto.</p>

Demonstração de Parágrafo

Este é o primeiro parágrafo de texto de exemplo.

Este é um segundo parágrafo. Note que ele está separado do anterior.

<span>
Container genérico em linha para marcar parte de um texto para estilização.
Texto <span style="color:red">destacado</span> em vermelho.

Demonstração de Span

Este texto tem uma parte importante que foi estilizada com a tag span.

<strong>
Indica que o texto tem forte importância (normalmente exibido em negrito).
<strong>Atenção:</strong> Isso é importante!

Demonstração de Strong

Atenção: esta é uma mensagem muito importante.

<em>
Indica ênfase no texto (normalmente exibido em itálico).
Eu <em>realmente</em> quero isso.

Demonstração de Ênfase

Eu realmente preciso que você se lembre disso.

<br>
Insere uma quebra de linha única.
Primeira linha<br>Segunda linha

Demonstração de Quebra de Linha

Endereço: Rua dos Gatos, 123
Cidade dos Cães, SP

<hr>
Cria uma linha horizontal temática (normalmente usada para separar conteúdo).
Seção 1<hr>Seção 2

Demonstração de Linha Horizontal

Conteúdo da Seção 1.

Conteúdo da Seção 2.

<mark>
Destaca texto como se tivesse sido marcado com um marcador amarelo.
Texto com <mark>destaque</mark> importante.

Demonstração de Destaque

Lembre-se de comprar leite e ovos na sua próxima visita.

<small>
Define texto menor (como letras miúdas ou comentários secundários).
Preço: R$100 <small>(sem impostos)</small>

<del>
Texto que foi deletado (normalmente riscado).
Preço: <del>R$200</del> R$150

Demonstração de Texto Deletado

<ins>
Texto que foi inserido (normalmente sublinhado).
Versão <ins>2.0</ins> lançada hoje.

Demonstração de Texto Inserido

O documento foi atualizado para a versão 3.5.

<sub>
Texto subscrito (mais baixo que o normal).
H<sub>2</sub>O

Demonstração de Subscrito

A fórmula da água é H2O.

<sup>
Texto sobrescrito (mais alto que o normal).
x<sup>2</sup> + y<sup>2</sup>

Demonstração de Sobrescrito

A equação é x2 + y2 = z2.

<blockquote>

<blockquote cite="...">Citação importante aqui.</blockquote>

<q>

<abbr>
Define uma abreviação ou acrônimo.
<abbr title="HyperText Markup Language">HTML</abbr>

Demonstração de Abreviação

Aprender HTML é o primeiro passo para o desenvolvimento web.

<code>

Use <code>console.log()</code> para debugar.
