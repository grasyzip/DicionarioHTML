Listas e Tabelas

Estas tags são usadas para criar listas ordenadas, não ordenadas e tabelas em páginas HTML.

Nenhuma tag encontrada com ""

Tente outro termo ou verifique a ortografia.

<ul> e <li>
Define uma lista não ordenada (com marcadores) e seus itens.
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

Demonstração de Lista Não Ordenada

  • Primeiro item da lista
  • Segundo item da lista
  • Terceiro item da lista
<ol> e <li>
Define uma lista ordenada (com números ou letras) e seus itens.
<ol>
  <li>Primeiro item</li>
  <li>Segundo item</li>
</ol>

Demonstração de Lista Ordenada

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo
<dl>, <dt>, <dd>
Define uma lista de descrição (<dl>), com termos (<dt>) e descrições (<dd>).
<dl>
  <dt>HTML</dt>
  <dd>Linguagem de marcação para páginas web</dd>
</dl>

Demonstração de Lista de Descrição

HTML
Linguagem de marcação para a estrutura da página.
CSS
Linguagem de estilos para a aparência da página.
<table>, <tr>, <td>, <th>
Define uma tabela, com linhas (<tr>), células de cabeçalho (<th>) e células de dados (<td>).
<table>
  <tr><th>Nome</th><th>Idade</th></tr>
  <tr><td>João</td><td>30</td></tr>
</table>

Demonstração de Tabela

Nome Idade País
Maria 25 Brasil
Carlos 42 Portugal
