Formulários

Estas tags são usadas para criar formulários interativos em páginas HTML.

Nenhuma tag encontrada com ""

Tente outro termo ou verifique a ortografia.

<form>
Container para elementos de formulário. Define onde os dados serão enviados.
<form action="/processar" method="post">...</form>

Demonstração de Formulário

Este é um formulário de exemplo.




<input>
Campo de entrada de dados. O tipo é definido pelo atributo `type`.
<input type="text" name="nome">
<input type="email" name="email">
<input type="password" name="senha">
<input type="submit" value="Enviar">

Demonstração de Campos de Input






<textarea>
Área de texto multilinha para entrada de texto livre.
<textarea name="mensagem" rows="4" cols="50">...</textarea>

<button>
Define um botão clicável. Pode ter tipos `submit`, `reset` ou `button`.
<button type="button">Clique aqui</button>

Demonstração de Botão

<select> e <option>
Cria uma lista dropdown de opções selecionáveis.
<select name="cidade">
  <option value="sp">São Paulo</option>
  <option value="rj">Rio de Janeiro</option>
</select>

Demonstração de Lista Suspensa


<label>
Define um rótulo para um elemento de formulário, melhorando a acessibilidade.
<label for="nome">Nome:</label> <input type="text" id="nome">

Demonstração de Rótulo

Clique no texto "Nome" para focar no campo de input.

<fieldset> e <legend>
Agrupa elementos de formulário relacionados e fornece um título para o grupo.
<fieldset>
  <legend>Informações Pessoais</legend>
  ...
</fieldset>

Demonstração de Agrupamento

Informações de Contato


<datalist>
Fornece uma lista de opções pré-definidas para um campo de input de texto.
<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
</datalist>
